# Test Cases for Movie Recommendation System

## Test Case 1: Valid Input (Happy Path)

### movies.txt
```
The Dark Knight,TDK123
action,thriller,drama
Inception,I456
action,scifi,thriller
The Godfather,TG789
crime,drama
Toy Story,TS012
animation,comedy,family
Interstellar,I345
scifi,drama,adventure
```

### users.txt
```
Ahmed Hassan,123456789
TDK123,I456
Sara Mohamed,987654321
TG789,TS012
Omar Ali,111222333
I456,I345
```

### Expected Output (recommendations.txt):
```
Ahmed Hassan, 123456789
Inception, Interstellar, The Godfather
Sara Mohamed, 987654321
The Godfather, Toy Story
Omar Ali, 111222333
The Dark Knight, Inception, Interstellar, The Godfather
```
